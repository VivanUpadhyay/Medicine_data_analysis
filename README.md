#  Medicine Data Analysis

##  Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a pharmaceutical dataset containing information about medicines, manufacturers, compositions, side effects, medical uses, and customer review percentages.

The objective is to discover meaningful insights, understand customer satisfaction patterns, identify leading pharmaceutical manufacturers, and visualize important trends using Python.

---

##  Dataset Information

- **Total Records:** 11,825
- **Manufacturers:** 759
- **Medicine Compositions:** 3,358
- **Medical Uses:** 712

### Dataset Features

- Medicine Name
- Composition
- Uses
- Side Effects
- Manufacturer
- Excellent Review %
- Average Review %
- Poor Review %

##  Exploratory Data Analysis

The project answers several important questions such as:

- Which manufacturers produce the most medicines?
- What are the most common medicine compositions?
- Which side effects occur most frequently?
- How are customer reviews distributed?
- What relationships exist between review metrics?

##  Key Insights

- The dataset contains over **11,800 medicines** from **759 manufacturers**.
- Sun Pharmaceutical Industries Ltd contributes the highest number of medicines.
- Excellent Reviews dominate customer feedback, indicating generally positive user experiences.
- Medicine compositions vary significantly, demonstrating a wide range of pharmaceutical formulations.
- Common side effects are shared across multiple medicines, revealing recurring adverse reactions.
- Most medicines receive moderate to high customer satisfaction scores.
- The correlation analysis shows a strong inverse relationship between Excellent Review % and Poor Review %.
- Only a small number of medicines appear as outliers in customer review distributions.

---

##  Conclusion

This analysis provides valuable insights into the pharmaceutical industry by examining medicine manufacturers, compositions, customer reviews, and side effects. The findings demonstrate how exploratory data analysis can uncover meaningful patterns in healthcare data and support data-driven decision-making.
